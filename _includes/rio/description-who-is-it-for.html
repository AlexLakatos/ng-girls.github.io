Você, mulher, está interessada em aprender a construir aplicativos para a web usando as tecnologias mais usadas do mercado? Temos uma boa notícia para você: estamos realizando uma oficina de um dia para iniciantes e ela irá acontecer num sábado, dia 13 de janeiro de 2018, no Rio de Janeiro!

Nós acreditamos que o mercado de TI irá se beneficiar muito ao trazer mais mulheres para a área de tecnologia. Por conta disso e também em acreditar na diversidade e na igualdade de gênero, estamos oferecendo uma oportunidade para as mulheres aprenderem a programar e se tornarem programadoras.

Durante o workshop você irá aprender a programar a sua primeira aplicação web em Angular!. Angular é uma plataforma open-source para a construção de aplicativos web, desenvolvida pelo Google. Ela oferece várias ferramentas para criar facilmente um website, plataforma digital, programas de computador e aplicativos para dispositivos móveis.

Para participar do workshop você precisa ter noção ou conhecimento básico em HTML, CSS e Javascript ou qualquer outra linguagem de programação. Se você iniciante e não conhece os itens listados acima, não tem problema. Vamos te oferecer total suporte e apoio para você conseguir desenvolver sua primeira aplicação em Angular.

O workshop é 100% gratuito. Para participar, preencha este formulário.
